Traffic light classifier model file (.hdf5) should be placed in this folder.
The current working model is:

ResNet50-UdacityRealandSimMix-Best-val_acc-1.0.hdf5

Compressed split-up model files can be extracted using 7-zip by the following commands:

sudo apt-get install p7zip-full
7z x ResNet50-UdacityRealandSimMix-Best-val_acc-1.0.hdf5.7z.001
